Hexo + Github Page 建置個人 Blog - (2) 串接 Hexo 與 Github Page

發表於2022-10-14|更新於2022-10-14
|閱讀量:

前情提要

前面已經弄完了 Hexo 的環境，接下來我們要來開始把他掛上去 Github Page ㄌ

弄弄你的 Github

應該不用寫怎麼創 Github 帳號對ㄅ，那我就直接開始建置 Github Page 搂~

創建新的 Repositories


點到這個畫面以後應該能看到右上角有一個綠綠的 New 對ㄅ，點下去。

設定 Repositories


- 看到紅色框框了對吧，在裡面輸入 (用戶名稱).github.io
> 這裡有硬性規定，一定要輸入這樣才會被識別為 Github Page
- 輸入完之後就直接點下面的

建立 ssh key(如果之前有用過就可以跳過)

這一步是因為 Hexo 在丟東西上去 Github 是透過 ssh 的方式。

  • 設定本地的 git 帳號
  • 產生 ssh key 在這裡他會跳一系列要輸入的東西，直接按 Enter 到底就可以。
  • 驗證 ssh 如果跳出類似圖片的內容就代表 OK ㄌ

把你的 ssh key 丟上去 Github

  • 打開 Github 的 Settings
  • 點選 SSH and GPG keys
  • 然後按下那個酷酷的 New SSH key
  • 進到這個畫面之後切回去 cmd
  • 獲取你的 ssh key
  • 把剛剛 cat 出來的東西全部複製貼到剛剛的 key 欄位去
  • 然後 Title 隨便設，接著按下 Add SSH key 就好

設定 hexo 讓他能把網頁 push 到 github

  • 打開前面配置好的資料夾中的 _config.yml
  • 滑到最底下改成
  • 配置好以後應該會長這樣
