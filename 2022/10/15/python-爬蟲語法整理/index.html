HTML

find()

搜尋第一個符合的 html 節點

find_all()

搜尋所有符合的 HTML 內容，可加關鍵字參數指定屬性並限制內容數量。

多標籤可串列

select_one()

節點下只有單個節點:#h3底下只有一個< a >
選取特定子節點
(放置於print中)

select()

節點下有多個節點:div 下有多個 a
選取子節點

CSS

find()

搜尋第一個符合的HTML及CSS屬性的節點

find_all()

搜尋所有符合的HTML與CSS屬性節點

select()

只使用CSS搜尋

find_parent() / find_parents()

向上搜尋節點

find_previous_siblings()

搜尋同階層的前一個節點

find_next_siblings()

搜尋同階層的後一個節點

取得itemprop屬性
